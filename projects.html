Notable Projects

Mem Scanner
A python tool made for simple application memory manipulation, based on CheatEngine. The goal of this project is to understand how to read and manipulate memory data, addresses and the memory structure of programs.
LOL Assistant
A python tool for assisting players in the game "League of Legends". It uses scrapping to collect data and then presents them to assist players on their gaming experience.
Bone Labelling Demo
This is a demo implementation of the network design proposed in my thesis titled "Deep Persistent Bone Labelling for Efficient Character Animation" written in Python. This implementation leverages LSTM neural networks and the Attention mechanism to achieve persistent and reliable bone identification across complex skeletal structures, enabling more realistic and consistent animations.
Object Collision Demo
A project developed to simulate object generation and collisions as part of my university Graphics course. The goal was to understand how to use shaders to create simple scenes and apply basic physics rules for realistic interactions.
