Σημαντικά Πρότζεκτ

Mem Scanner
Ένα εργαλείο ανεπτυγμένο στην γλώσσα Python για την διαχείρηση μνήμης προγραμμάτων, βασισμένο στην ιδέα της εφαρμογής CheatEngine. Στόχος της εφαρμογής είναι η κατανόηση της διαχείρισης της δομής, των διευθύνσεων και των δεδομένων της μνήμης των προγραμμάτων.
LOL Assistant
Ένα εργαλείο ανεπτυγμένο στην γλώσσα Python για να προσφέρει βοήθεια στους παίκτες του παιχνιδιού "League of Legends". Χρησιμοποιεί scraping για την συλλογή δεδομένων και τα παρουσιάζει στους παίκτες για να βελτιώσει την εμπειρία τους.
Bone Labelling Demo
Αυτή είναι μια δοκιμαστιή υλοποίηση του δικτύου που αναφέρω στη διπλωματική μου με τίτλο "Βαθιά επίμονη σήμανση οστών για αποτελεσματική κίνηση χαρακτήρων" ανεπτυγμένο σε Python. Αυτή η υλοποίηση αξιοποιεί τα Νευρωνικά Δίκτυα LSTM και τον Μηχανισμό Προσοχής για την επίτευξη επίμονης και αξιόπιστης αναγνώρισης οστών σε περίπλοκες σκελετικές δομές, επιτρέποντας πιο ρεαλιστικές και συνεπείς κινούμενες εικόνες.
Object Collision Demo
Μία εργασία που αναπτύχθηκε για την προσομοίωση δημιουργίας αντικειμένων και συγκρούσεων ως μέρος του πανεπιστημιακού μαθήματος Γραφικών. Ο στόχος ήταν να κατανοήσουμε πώς να χρησιμοποιούμε shaders για να δημιουργήσουμε απλές σκηνές και να εφαρμόζουμε βασικούς κανόνες φυσικής για ρεαλιστικές αλληλεπιδράσεις.
