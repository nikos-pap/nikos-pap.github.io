Γλώσσες Προγραμματισμού

Python (Επιδέξιος)

Η Python είναι η πρώτη γλώσσα προγραμματισμού που έμαθα και τη χρησιμοποιώ συχνά λόγω της απλότητάς της και της ταχύτητας με την οποία μπορώ να γράφω σενάρια για διάφορες εργασίες. Έχω εφαρμόσει εκτενώς αντικειμενοστραφή προγραμματισμό (OOP) στα έργα μου, χρησιμοποιώντας κλάσεις για τη δημιουργία δομημένου και συντηρήσιμου κώδικα. Με την πάροδο του χρόνου, έχω εργαστεί με μια ποικιλία βιβλιοθηκών, συμπεριλαμβανομένων των bs4, json και requests για scraping και αιτήματα σε API, OpenCV για υπολογιστική όραση, NumPy για αριθμητικούς υπολογισμούς, TensorFlow και PyTorch για νευρωνικά δίκτυα, Pymem για χειρισμό μνήμης, καθώς και Tkinter για εφαρμογές GUI. Το εκτεταμένο περίβαλλον και η ευελιξία της Python την καθιστούν, για εμένα, τη γλώσσα για γρήγορη ανάπτυξη και αποτελεσματικές λύσεις.

Java (Προχωρημένος)

Χρησιμοποίησα την Java εκτενώς κατά τη διάρκεια της φοίτησής μου στο πανεπιστήμιο, όπου έγινε το θεμέλιο για την εκμάθηση πολλών βασικών εννοιών προγραμματισμού, ιδιαίτερα του Αντικειμενοστρεφούς Προγραμματισμού. Η Java ήταν το κλειδί για την κατανόηση και την εφαρμογή διαφόρων μοτίβων σχεδίασης και ήταν η γλώσσα που χρησιμοποίησα για να κατανοήσω τις δομές δεδομένων όπως λίστες, ουρές, δέντρα και άλλα. Εργάστηκα επίσης με βιβλιοθήκες όπως η FreeTTS για να ενσωματώσω τη λειτουργία μετατροπής κειμένου σε ομιλία. Αν και δεν έχω χρησιμοποιήσει πολύ την Java σε πολύπλοκα προσωπικά προτζεκτ, παραμένει η γλώσσα όπου έχω το βαθύτερο θεωρητικό υπόβαθρο, χάρη στην ολοκληρωμένη εκπαίδευση και την πρακτική εμπειρία που έχω αποκτήσει σε αυτήν.

C (Προχωρημένος)

Έχω χρησιμοποιήσει την C κυρίως για προγραμματισμό σε επίπεδο συστήματος, όπου έχω αναπτύξει μια ισχυρή κατανόηση του πολυνηματισμού και της πολυδιεργασίας χρησιμοποιώντας τις βιβλιοθήκες pthread και OpenMP, εφαρμόζοντας μηχανισμούς συγχρονισμού όπως mutexes για τη διασφάλιση της ασφάλειας των νημάτων για ταυτοχρονισμό. Η εμπειρία μου περιλαμβάνει εργασίες σε βασικές αρχές λειτουργικού συστήματος UNIX, όπως ο χρονοπρογραμματισμός διεργασιών και η επικοινωνία μέσω κλήσεων συστήματος και πυρήνα. Η C με βοήθησε να κατανοήσω τα βασικές των έννοιες προγραμματισμού σε επίπεδο συστημάτων και τον τρόπο αλληλεπίδρασης των συστημάτων με το υλικό, καθιστώντας το ένα πολύτιμο εργαλείο στο σύνολο των δεξιοτήτων μου.

HTML/CSS/JS (Ικανός)

Έχω μια θεμελιώδη κατανόηση της ανάπτυξης στατικών σελίδων, με την χρήσηη HTML, CSS και JavaScript. Αυτές οι βασικές τεχνολογίες μου επιτρέπουν να δημιουργήσω βασικές ιστοσελίδες και διεπαφές. Επιπλέον, έχω κάποια εμπειρία με σύγχρονα πλαίσια JavaScript όπως το ReactJS και το AngularTS για δυναμικές εφαρμογές. Κατά τη διάρκεια της πρακτικής μου, εργάστηκα με Angular και με την βιβλιοθήκη WebRTC για επικοινωνία σε πραγματικό χρόνο, βελτιώνοντας περαιτέρω την κατανόησή μου για τις δυναμικές εφαρμογές Διαδικτύου. Ενώ η εμπειρία μου στην ανάπτυξη ιστοσελίδων είναι σε βασικό επίπεδο, είμαι εξοικειωμένος με βασικές έννοιες και εργαλεία για τη δημιουργία και τη συντήρηση λειτουργικών ιστοσελίδων.

Unity (Αρχάριος)

Οι γνώσεις μου για το Unity είναι σε αρχάριο επίπεδο, έχοντας το χρησιμοποιήσει για δύο πανεπιστημιακές εργασίες. Η πρώτη ήταν το Object Collision Demo, όπου έμαθα την δημιουργία σωματίδιων, εφάρμοσα κανόνες φυσικής και δούλεψα με υφές και βασικά 3D αντικείμενα. Στη δεύτερη εργασία, μια συνεργατική δοκιμαστίκη προσομοίωση ενός Ψηφιακού Μουσείου με μια ομάδα πολλών ατόμων, ασχολήθηκα με τη δημιουργία πολλαπλών σκηνών, τον σχεδιασμό απλών μενού και την χρήση Mesh αντικειμένων. Μία από τις σημαντικές συνεισφορές μου εκεί ήταν η δημιουργία προβολής μοντέλων που διέθετε διαδραστική λειτουργία περιστροφής και ζουμ. Αυτά τα έργα μου έδωσαν πρακτική εμπειρία με τα ισχυρά εργαλεία της Unity για τη δημιουργία καθηλωτικών τρισδιάστατων περιβαλλόντων.
